## üí° Project Goal

The primary objective of this project was to transform raw retail transaction data into actionable business intelligence. Specifically, the analysis focused on:

1.  Identifying **top-performing product categories** and their contribution to total sales.
2.  Analyzing **sales distribution** across different customer demographics (Gender, Generation).
3.  Determining **sales volume** by geographic region and product type.
